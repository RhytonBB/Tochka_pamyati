# Инструкция по локальному запуску проекта «Точка памяти»

Этот проект состоит из бэкенда на Go и фронтенда на Vue 3. Для хранения геоданных используется PostgreSQL с расширением PostGIS.

## Предварительные требования
- [Go 1.21+](https://go.dev/dl/)
- [Node.js 18+](https://nodejs.org/)
- **База данных**: PostgreSQL 15+ с расширением PostGIS.

---

## Способ 1: Ручная установка (если Docker не работает)

### 1. Установка PostgreSQL и PostGIS
1. Скачайте и установите **PostgreSQL** (версии 15 или выше) с официального сайта: [postgresql.org](https://www.postgresql.org/download/windows/).
2. В конце установки (или через Stack Builder) выберите установку расширения **PostGIS**. Оно находится в разделе `Spatial Extensions`.
3. Запомните пароль суперпользователя (`postgres`), который вы укажете при установке.

### 2. Создание базы данных
Откройте **pgAdmin 4** (устанавливается вместе с PostgreSQL) или используйте консоль `psql`:
1. Создайте базу данных с названием `memory_point`.
2. Выполните в этой базе следующий SQL-запрос для включения гео-функций:
   ```sql
   CREATE EXTENSION IF NOT EXISTS postgis;
   ```

### 3. Настройка окружения
В папке `backend` откройте файл `.env` и укажите данные вашего локального PostgreSQL:
```env
DATABASE_URL=postgres://postgres:ВАШ_ПАРОЛЬ@localhost:5432/memory_point?sslmode=disable
```

## Способ 2: Через Docker Desktop (если работает)
Если вы все же решите использовать Docker (например, через WSL2), в корне проекта выполните:
```bash
docker-compose up -d
```

---

## Шаг 2: Запуск бэкенда
1. Перейдите в директорию `backend`:
   ```bash
   cd backend
   ```
2. Убедитесь, что файл `.env` настроен (см. выше).
3. Установите зависимости и запустите сервер:
   ```bash
   go mod tidy
   go run cmd/server/main.go
   ```
   Бэкенд будет доступен по адресу `http://localhost:8080`.

---

## Шаг 3: Запуск фронтенда
1. Перейдите в директорию `frontend`:
   ```bash
   cd frontend
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите проект в режиме разработки:
   ```bash
   npm run dev
   ```
   Фронтенд будет доступен по адресу `http://localhost:5173`.

---

## Структура проекта
- `/backend` — API на языке Go (Echo framework).
- `/frontend` — Клиентское приложение на Vue 3 + Tailwind CSS + MapLibre GL JS.
- `docker-compose.yml` — Конфигурация для быстрого запуска базы данных.

## Основные функции
- Просмотр карты памятников ВОВ.
- Подача сигналов об угрозах разрушения или повреждения объектов.
- Просмотр детальной информации о памятниках и текущих угрозах.
